## Date

## Objective
The objective of the PalindromeChecker App is to design and implement a console-based Java application that validates whether a given string is a palindrome under different conditions, while strengthening core programming fundamentals and data structure concepts.

## Goal
Compare the performance of different palindrome checking approaches.

## Flow
1. Run multiple palindrome algorithms:
   - Iterative char array check
   - Stack-based check
   - Deque-based check
2. Capture execution time using `System.nanoTime()`.
3. Display results to compare performance.

## Key Concepts Used
- **System.nanoTime()** – Measure execution time in nanoseconds.
- **Algorithm Comparison** – Compare efficiency of different approaches.
